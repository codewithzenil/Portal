# **App Name**: CodeWithZenil

## Core Features:

- Public Homepage: Display a landing page with school branding, a certificate verification button, and institutional information.
- Certificate Verification: Allow users to verify certificates by entering a Certificate ID or Student Roll Number and displaying the certificate details if found.
- Admin Dashboard: Provide a secure, password-protected admin interface for managing certificate records.
- Certificate Management: Enable admin users to add new certificates, upload certificate PDFs, edit or delete records, and export the database content.
- Google Sheets Integration: Sync certificate records with a Google Sheet, automatically updating when changes are made in the admin dashboard.
- QR Code Generation: Auto-generate a QR code for each certificate that links to the verification page with the certificate ID, displaying the QR code on the certificate screen for download.
- Email Notifications (Optional): Utilize a tool to draft and send email notifications to students when their certificates are added to the system.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and academic authority.
- Background color: Light blue (#E8EAF6), a very pale desaturation of the primary, for a clean, academic look.
- Accent color: Purple (#7E57C2) to highlight calls to action and important information.
- Body and headline font: 'Inter', a sans-serif with a modern and neutral appearance.
- Use simple, clear icons to represent actions and information.
- Implement a clean card-based UI for certificate displays and form inputs.
- Incorporate subtle fade and transition effects for a smooth user experience.
